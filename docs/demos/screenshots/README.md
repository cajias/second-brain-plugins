# Demo Screenshots

Sample output from the `karpathy-llm-wiki` plugin, captured from a real Obsidian vault
(`~/Documents/Obsidian Vault/llm-wiki/`). All assets below are auto-generated by the
plugin's `kb-compile` / analytics pipeline — these are real pipeline outputs, not mockups.

## Files

| File | Demonstrates |
|------|--------------|
| `tag-distribution.png` | Tag-frequency chart produced by the analytics step, showing how notes cluster by tag. |
| `knowledge-type-distribution.png` | Breakdown of notes by `knowledge_type` (pattern, fact, correction, design — the taxonomy actually used in this vault). |
| `growth-over-time.png` | Vault-growth time series showing note-creation cadence. |
| `health-summary.png` | Vault-health snapshot emitted by the analytics pipeline. |
| `example-stats-dashboard.md` | Auto-generated stats markdown written to `wiki/_meta/stats.md`. Quotable in docs. |

## Regenerating

These outputs are produced by running `kb-compile` (or the analytics subcommands) against
a populated vault. See `plugins/karpathy-llm-wiki/` for the pipeline that creates them.
